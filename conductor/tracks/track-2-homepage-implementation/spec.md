# Spec: Track 2 - Homepage Implementation

## 目标
实现一个高转化率、体现专业解决能力与负责任态度的响应式首页。

## 页面区块设计

### 1. Hero Section (重构)
- **视觉**: 使用高质量的物流背景（占位），配合深色叠加层提升文字可读性。
- **文案**: 
  - 主标题：“把冷链交给我们，您只需要关心生意”
  - 副标题：突出“解决问题的能力”与“负责任的态度”（全程制冷、异常快处、保险兜底）。
  - CTA: “获取专业冷链解决方案”。

### 2. Services Section
- **布局**: 网格布局展示核心业务。
- **项**: 港口冷藏集装箱运输、暂落箱/插电、内装仓储、多式联运。
- **强调**: 每个服务带有一个简短的“能力证明”（如：熟悉上港规则，快速换箱）。

### 3. Core Values (解决问题的态度)
- **设计**: 使用大图标或卡片展示。
- **要点**: 
  - 责任不外包（自营运作）
  - 过程可追溯（GPS/云监控）
  - 风险有兜底（高额保险）

### 4. Clients/Trust
- **设计**: 合作伙伴/服务客户的 Logo 墙（灰度处理，悬停恢复彩色）。

## 技术要求
- 所有的图片使用 `next/image`。
- 保证移动端首屏加载时间。
- 使用 Tailwind 4 的容器类与响应式前缀。
